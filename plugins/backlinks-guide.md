## 快速操作指南

### 第一批：用 Google 直接登录（26 个）
```bash
# 批量操作 — 每个平台登录后添加你的网站链接
xbrowser seo login --platform youtube --cdp http://localhost:9221
xbrowser seo login --platform medium --cdp http://localhost:9221
xbrowser seo login --platform pinterest --cdp http://localhost:9221
xbrowser seo submit-backlink --platform youtube --cdp http://localhost:9221
xbrowser seo submit-backlink --platform medium --cdp http://localhost:9221
# ... 以此类推
```

### 第二批：用 GitHub 直接登录（3 个）
```bash
xbrowser seo login --platform github --cdp http://localhost:9221
xbrowser seo login --platform stackoverflow --cdp http://localhost:9221
```

### 第三批：无登录直接提交（3 个）
```bash
xbrowser seo submit-guest-post --platform css-tricks --cdp http://localhost:9221 \
  --name "Your Name" --email "you@example.com" --topic "Your article pitch"
```
